# Web Server Tech Challenge 2025

## How to run

Requirements:
* Linux
* Docker
* OpenSSL
* Ansible

Clone and run ```./auto_setup.sh```
Open your browser and navigate to ```https://localhost```
There might be a security warning, this is because it's self signed.

Linux is listed as a requirement, but this was built on WSL 2.
Windows users will need to install WSL 2 and Docker Desktop, then enable its WSL 2 backend.
